Grammar

As mentioned above, the Markup grammar defines a mapping between a Markup document and an abstract syntax tree. The tree is built out of tagged elements and strings.

Markup was originally developed in Lisp where the obvious representation for a Markup document is as s-expressions, with each tree represented by a list whose first element is a symbol indicating the treeâ€™s tag.
